Student Attendance Tracking System

This program is a simple student attendance system that I created using Python. It helps to store and manage student details like name, course, and attendance.

At the beginning, I created an empty list called students to save all the student records, and I used a variable student_id to give each student a unique ID.

When the user chooses the add option, the program asks for student details and saves them in the list. Each time a new student is added, the ID increases automatically.

The update option allows the user to change a student’s course and attendance using their ID. If the ID is not found, it shows a message.

The show option displays all the student records in a simple format, and if there is no data, it shows “No data”.

The average option calculates the average attendance of all students.

I also used a menu with a loop so the program keeps running until the user selects the exit option.



Object-Oriented Programming

This program is a simple banking system created using Python and basic OOP (Object-Oriented Programming). It has three main classes: Account, Customer, and Transaction.

The Account class is used to store account details like account number and balance. It also has functions to deposit money, withdraw money, and display the balance. When money is deposited, the balance increases, and when money is withdrawn, it checks if there is enough balance before subtracting.

The Customer class is used to store customer information like the name and their account. It also has a function to display the customer’s details along with their account balance.

The Transaction class is used to perform actions like deposit or withdraw. When a transaction is created, it automatically processes the transaction based on the type (deposit or withdraw).

In the testing part, I first created an account with an initial balance. Then I created a customer linked to that account. After that, I displayed the customer details and performed some transactions like depositing and withdrawing money. Finally, the updated balance is shown.
